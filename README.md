# ENSAM-Hub

ENSAM Focus Hub is a **web-based productivity platform** designed for students at ENSAM Casablanca. It combines tools like **Focus Mode (Pomodoro Timer), Notes, Drive Access, and other productivity features** in a clean, minimal interface. The platform emphasizes learning by doing and helps students manage their study sessions efficiently.

---

## Features

### Home Page
- Welcome message and overview.
- Quick access buttons to all tools (Notes, Pomodoro Timer, Drives, Other Tools).
- Bright/Dark theme toggle.

### Focus Mode
- **Pomodoro Timer**: 25-minute study sessions with countdown.
- **Stats Tracker**: Displays the number of completed sessions.
- **Session Completion Animation**: Visual feedback when a session ends.
- Bright/Dark mode toggle available.
- Clean, distraction-free interface.

### Notes (Placeholder)
- Placeholder page for taking notes (future implementation).

### Drives
- Quick links to Google Drive folders organized by semester.
- Students can easily access course materials and exercises.

### Other Tools
- Links to external tools like PDF converters, notebooks, and online learning platforms.

---

## Technologies Used

- **Python Flask** – Web framework to serve pages.
- **HTML5 & CSS3** – Page structure and styling.
- **JavaScript** – Interactive features (Pomodoro Timer, theme toggle, animations).
- **No backend database** – All tools currently run client-side or link to external services.
